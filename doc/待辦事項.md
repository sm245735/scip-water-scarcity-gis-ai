#### 空間座標（等康康提供）
- 等康康提供水庫空間座標檔 → 補填 reservoirs.lon / reservoirs.lat / reservoirs.geom

#### 氣象資料缺口（需進一步確認）
- CWB 開放資料平台：確認是否有網格化日均溫可下載（opendata.cwa.gov.tw）
- TCCIP：確認是否有其他氣象要素（含均溫的網格化產品）
- 水利署地面水觀測站：確認是否可空間內插氣溫至水庫位置
- 水利署水文情形網：確認水庫蒸發量觀測資料是否對外開放
- CWB 氣象站：確認直徑 20cm 蒸發皿資料是否可取得
- Hargreaves 蒸發量估算公式評估（需氣溫 + 濕度）
- 以上所有氣象資料需覆蓋 2016-2023 年

#### 科學園區用水量（竹科相關）
- 確認 EIA 能源局是否有科學園區用水量月資料可下載
- 向科學園區管理局申請或找公開年報
- 確認資料時間範圍（需與 LSTM 訓練期 2016-2023 對齊）
- 取得月資料後，用線性內插擴展為日尺度用水量估算
