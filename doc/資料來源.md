## 水庫即時水情 API

### 即時水情（新竹感興趣區域）

| 項目 | 內容 |
|------|------|
| **API URL** | `https://opendata.wra.gov.tw/api/v2/2be9044c-6e44-4856-aad5-dd108c2e6679` |
| **Swagger 文件** | https://opendata.wra.gov.tw/openapi/swagger/index.html |
| **更新頻率** | 每小時（即時） |
| **資料範圍** | 全台灣 49 個水庫 |
| **認證** | 無（公開 API） |
| **過濾參數** | `?reservoiridentifier=10201`（可指定水庫） |

### 欄位說明

| 欄位（英文） | 欄位（中） | 說明 |
|-------------|-----------|------|
| `observationtime` | 觀測時間 | ISO 8601 時間格式 |
| `reservoiridentifier` | 水庫 ID | 五碼數字（如 10201 = 寶山水庫） |
| `effectivewaterstoragecapacity` | 有效蓄水量 | 萬立方公尺 |
| `waterlevel` | 水位 | 公尺 |
| `inflowdischarge` | 入流量 | cms（立方公尺/秒） |
| `totaloutflow` | 總出流量 | cms |
| `accumulaterainfallincatchment` | 集水區累積降雨 | mm |
| `waterdraw` | 取水量 | cms |

## 歷史日資料取得方式

### 方法 A：手動下載（目前建議）
水利署統計頁面（一次查一天）：
`https://fhy.wra.gov.tw/ReservoirPage_2011/Statistics.aspx`

### 方法 B：水利署 OpenData（日後探索）
- `https://opendata.wra.gov.tw/api/v2/..._daily`
- `https://opendata.wra.gov.tw/api/v2/..._history`
（目前測試無效，留待日後確認）

### 方法 C：data.gov.tw
在 https://data.gov.tw 搜尋「水庫 日資料」或「水庫 蓄水 統計」
