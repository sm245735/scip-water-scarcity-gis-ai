## 水庫日資料

### Comparison API（主要歷史資料）
**URL:** `POST https://fhy.wra.gov.tw/Disaster/api/ReservoirHistoryApi/GetComparision`
**更新頻率：** 每日 1 次（離線批次收集）  
**內容：** 全台 20 個水庫的歷史日資料  
**參數：**
- `ST_NO`: 水庫 ID（最多 9 個，用逗號分隔）
- `StartDate`: 開始日期（YYYY/MM/DD）
- `EndDate`: 結束日期（YYYY/MM/DD）

**重要限制：**
- API 在日期範圍的「頭尾」各回傳 1 筆記錄
- **策略：每次查詢區間=1天**，即可取得完整每日資料
- 每天呼叫 3 次（20個水庫分 3 批）

### SOAP API（即時資料）
**URL:** `https://fhy.wra.gov.tw/ReservoirPage_2011/WS/wsReservoir.asmx/GetInfos`
**更新頻率：** 每小時（即時）
**內容：** 全台 103 個水庫的當日最新資料
**用途：** 每日自動化收集，逐步建立歷史資料庫

## 歷史資料取得

### 水庫日資料（2016-2026）
```bash
docker exec thesis_python_dev python /app/src/data_pipeline/水庫歷史日資料收集_v1_20260412.py \
  --start 2016/01/01 --end 2026/04/12 --days-per-batch 10
```
- 預計執行時間：6-8 小時（可背景執行）
- 每天自動收集：crontab 每天 03:00

### 即時水庫日資料（每日更新）
```bash
docker exec thesis_python_dev python /app/src/data_pipeline/水庫日資料收集_v1_20260412.py
```
