## 🚀 常用指令

### 啟動環境
```bash
cd /app
docker compose up -d
```

### 停止環境
```bash
docker compose down
```

### 進入 Python 容器
```bash
docker exec -it thesis_python_dev bash
```

### 執行 Python 腳本
```bash
docker exec thesis_python_dev python /app/你的腳本.py
```

### 查看容器日誌
```bash
docker compose logs -f
```

### 查看容器狀態
```bash
docker compose ps
```

### 重新建置容器
```bash
docker compose build --no-cache
docker compose up -d
```

## ⚠️ 常見問題

### Q: 容器啟動失敗？
```bash
# 檢查狀態
docker compose ps

# 查看日誌
docker compose logs
```

### Q: 資料庫連線失敗？
```bash
# 確認 PostgreSQL 健康狀態
docker exec thesis_postgres pg_isready -U sm245735
```

### Q: 需要重新安裝 Python 套件？
```bash
# 進入容器
docker exec -it thesis_python_dev bash

# 安裝新套件
pip install 套件名稱

# 更新 requirements.txt
pip freeze > requirements.txt
```

### Q: 資料不見了？
- 確認有使用 Volume：`postgres_data`
- 執行 `docker compose down` 不會刪除 Volume
- 執行 `docker compose down -v` 才會刪除（資料會不見！）
