## 📂 集水區 shapefile

- **來源：** 經濟部水利署《110年度全臺839子集水區範圍圖》
- **格式：** Shapefile（EPSG:3826，須轉 EPSG:4326 才能與 TCCIP 對齊）
- **數量：** 839 個子集水區（全臺灣）
- **存放位置：** `data/05_集水區/110年度全臺839子集水區範圍圖_UTF8.shp`

### 用途
1. 與 TCCIP 網格化降雨資料做 **Spatial Join**（ST_Intersects）
2. 計算各集水區範圍內的 TCCIP 網格平均日降雨量
3. 作為水庫入流量預測模型的 **Feature（自變數）**

## 📂 新竹縣市集水區（9個）

目前用於 LSTM 模型的新竹縣市集水區，共 **9 個**：

| 集水區名稱 | 說明 |
|-----------|------|
| **頭前溪** | 寶山第二水庫（蓄水量3,100萬噸）的主要水源集水區，**本研究重點** |
| 中港溪 | 新竹縣主要河流，流入竹北 |
| 大安溪 | 苗栗縣與臺中市界河 |
| 大甲溪 | 臺中市重要水系 |
| 客雅溪排水 | 新竹市區排水系統 |
| 新港溪 | 新竹縣沿海水系 |
| 淡水河 | 北部重要流域（桃園/新北/臺北） |
| 蘭陽溪 | 宜蘭縣主要河流 |
| 鹽港溪排水 | 新竹縣沿海排水 |

### 生成方式
1. 將 shapefile 匯入 PostGIS（EPSG:3826 → EPSG:4326）
2. 用 `ST_Intersects` 取出新竹縣市界內的集水區
3. 對每個集水區與 TCCIP 網格做 Spatial Join，取交集範圍內網格的加權平均降雨量
4. 輸出為 `data/新竹縣市_集水區每日降雨量_2023.csv`

### 與寶二水庫的關聯

```
頭前溪集水區降雨
       ↓
  寶山第二水庫蓄水量 ↑（延遲效應）
       ↓
  竹科供水風險 ↓
```

頭前溪集水區的降雨量是**最直接影響寶二水庫蓄水量的指標**，也是 LSTM 模型的核心 Feature。
